Запросы:
    TABLES - возвращает список таблиц в базе
    FIELDS - список полей в таблице
    CREATE TABLE <tablename>(<fieldtype1> <fieldname1>, <fieldtype2> <fieldname2>, ...)
        (типы: STRING / NUM)
    DROP TABLE <tablename>
    INSERT INTO <tablename> VALUES(<field1>, <field2>, ...)
    


в условии where можно использовать именя полей таблицы, числовые и строковые константы,
а также следующие операторы (по убыванию приоритета):

----------------------------------------------------------
операторы     |      допустимые типы операндов
----------------------------------------------------------
унарный "-"   |      числовые
*, /          |      числовые
+, -          |      числовые
>, >=, <, <=  |      числовые
=, !=, <>     |      любые, но один тип и слева и справа
унарный NOT   |      логические
AND           |      логические
OR            |      логические
----------------------------------------------------------

в результате выводятся только те строки таблицы, для которых выражение после WHERE вернуло True
