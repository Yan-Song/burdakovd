Инструкции по сборке.

Нужно установить cmake ("sudo aptitude install cmake")

затем создать директорию, к примеру ./Build/Release, перейти в неё
и выполнить: "cmake -DCMAKE_BUILD_TYPE=RELEASE ../.." (если созданная директория где-то в другом месте, то вместо "../.." пишем полный путь к директории с исходниками)
и затем "make"

при последующих изменениях кода только "make"

для Debug конфигурации создать отдельную директорию и аналогично.
